üß† Algorithm Explanation

The autoplay uses a sophisticated evaluation function that:

  1. Simulates each possible move (up, down, left, right)
  2. Evaluates the resulting board state using multiple criteria
  3. Selects the move with the highest score based on:
    • Immediate score gain from merging tiles
    • Number of empty cells (more is better)
    • Monotonic arrangement (tiles increase/decrease in order)
    • Smooth transitions between adjacent tiles
    • Bonus for keeping the highest tile in a corner

This strategy typically achieves the 2048 tile and often reaches 4096 or higher!
